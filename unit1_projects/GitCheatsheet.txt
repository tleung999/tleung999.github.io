Git Cheatsheet
Need Help?
		git help <command>
		example git help config

Create a new user
		git config -global user.name "your name"

Add email to user
		git config -global user.email "your email"

Add local directory for git to track
		cd to directory
		git init

Check status
		git status

See commit log
		git log

Add files to staging area
		git add --all
		git add .
		git add <filename>
		git add *.txt /* add all files with txt extention*/
		git add docs/ /* add all files in doc directory*/
		git add "*.txt" /* add all txt files in the whole project*/

Commit locally
		git commit -m "some message"  (-m switch for message)

Push files to a remote repository
		git push origin master

Fetch Changes 
		git remote -v /*check to make sure your remotes are setup correctly*/
		git fetch upstream /*fetch updates from the remote called upstream*/
		git branch /*check your current branch*/
		git checkout master /*Use this only if master is not the current branch*/
		git merge upstream/master -m "some msg" /*this will merge the remote upstream/master to your master repo*/
		git commit -m "some msg" /*this will commit the merge*/
